# Homepage

A personal portfolio and homepage built with [Hugo](https://gohugo.io/) and [Tailwind CSS](https://tailwindcss.com/).

## Features

- **Portfolio:** Showcases projects and work.
- **About:** Information about the author.
- **Homelab:** Details about personal server setups.
- **Tools:** A list of frequently used tools and resources.

## Development

### Prerequisites

- [Hugo](https://gohugo.io/installation/)
- [Node.js](https://nodejs.org/) and npm

### Setup

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd homepage
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the Hugo development server:
   ```bash
   hugo server
   ```

### Building for Production

To generate the static site in the `public/` directory:

```bash
hugo
```

## Project Structure

- `assets/`: CSS and images.
- `content/`: Markdown files for site pages.
- `layouts/`: Hugo templates.
- `static/`: Static assets (images, etc.).
